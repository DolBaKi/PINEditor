# PIN_Editor

This Project is made to edit KartRider.pin file. I used Viual Studio 2019 and .Net Framework 4.8. Now, This is just a prototype.

## Functions

- open KartRider.pin
- edit KartRider.pin(but this is not stable)
- save KartRider.pin
